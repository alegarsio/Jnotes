JNote is a professional, web-based notebook environment specifically designed for the Jackal programming language. It provides a seamless interface for writing, executing, and visualizing data with a focus on a clean and vibrant user experience.

## Key Features

* **Vibrant Minimalist Interface**: A floating card-based design with glassmorphism effects and an elegant white color scheme to minimize eye strain during long coding sessions.
* **Professional Code Editor**: Features automatic line numbering, auto-indentation,
* **Super-Detailed Data Visualization**: Deep integration with ECharts supporting line, area, bar, and pie charts with advanced zoom, panning, and image export capabilities.
* **Automated File Management**: An intelligent autosave system that ensures your progress is secured without manual intervention.
* **Last-Opened Session Memory**: The environment automatically remembers and restores the last file you were working on after a page reload.
* **Polished Terminal Output**: A minimalist, integrated terminal that displays execution results clearly within the workspace.
